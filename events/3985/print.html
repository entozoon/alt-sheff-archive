Sharrow Lantern Carnival

Date, Time & Location Sun 12 Apr 2015, 7.30pm Mount Pleasant Park, Sharrow Lane, nr London Rd. S11 8AE
Details Following last year's "Space" theme, Sharrow Lantern Carnival returns to planet Sheffield with "Life on Earth" as 2015's theme. Anyone can create a lantern on that theme; if you need some inspiration check out the Pinterest board. The carnival gathers in Mount Pleasant park from 7.30pm, setting off at 8pm for a procession towards a celebration in the General Cemetery 9-10.30pm. @sharrowlanterns Creative Action Network
